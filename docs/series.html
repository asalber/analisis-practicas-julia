Ejercicio 6.3 Dibujar una gráfica con los 10 primeros términos de las series geométricas \(\sum r^n\) para \(r=-2\), \(r=-1/2\), \(r=1/2\) y \(r=2\). ¿Para qué valores de \(r\) crees que converge la serie?

Solución 
using Plots, LaTeXStrings
an = [(-2)^n for n = 0:9]
scatter(0:9, cumsum(an), label=L"$\sum (-2)^n$")

La serie diverge.

bn = [(-1/2)^n for n = 1:9]
scatter(0:9, cumsum(bn), label=L"$\sum (-1/2)^n$")

La serie converge.

cn = [(1/2)^n for n = 0:9]
scatter(0:9, cumsum(cn), label=L"$\sum (1/2)^n$")

La serie converge.

dn = [2^n for n = 0:9]
scatter(0:9, cumsum(dn), label=L"$\sum 2^n$")

La serie diverge.

Se puede concluir que la serie converge para \(|r|<1\).

Ejercicio 6.4 Dibujar en la misma gráfica los 100 primeros términos de las series \(\sum \frac{1}{n^p}\) para \(p=1/2\), \(p=1\), \(p=3/2\) y \(p=2\).

¿Para qué valores de \(p\) crees que la serie converge?

Solución 
using Plots, LaTeXStrings
a(n,p) = 1/n^p
an = [a(n,1/2) for n = 1:100]
scatter(cumsum(an), label=L"$\sum \frac{1}{p^{1/2}}$", legend=:topleft)
bn = [a(n,1) for n = 1:100]
scatter!(cumsum(bn), label=L"$\sum \frac{1}{p}$")
cn = [a(n,3/2) for n = 1:100]
scatter!(cumsum(cn), label=L"$\sum \frac{1}{p^{3/2}}$")
dn = [a(n,2) for n = 1:100]
scatter!(cumsum(dn), label=L"$\sum \frac{1}{p^2}$")

Se puede concluir que la serie converge para \(p>1\).

Ejercicio 6.5 Una suma parcial de cualquier serie convergente se puede utilizar como aproximación de la suma de la serie. La aproximación será mejor cuanto mayor sea el orden de la serie parcial, pero depende de la velocidad a la que la serie converja a su suma. En el caso de las series alternadas que converjan, el error en la estimación de la suma mediante la suma parcial de orden \(n\) siempre es menor o igual que el término de \(n\) de la sucesión correspondiente, es decir, para la serie \(\sum a_n\), se cumple que

\[ |\sum_{n=1}^\infty a_n - \sum_{i=1}^n a_i|\leq a_n \]

Calcular la suma aproximada de las siguientes series alternadas con un error menor de \(10^{-10}\). ¿Cuál es la primera suma parcial que da esa aproximación?

  1. \(\sum \frac{(-1)^n}{n!}\)
Solución 
error = 10^-10
a(n) = (-1)^n/factorial(n)
i = 0
while abs(a(i)) >= error
    i += 1
end
println("Suma parcial de orden $i")
println("Aproximación: $(sum(a, 0:i))")
Suma parcial de orden 14
Aproximación: 0.36787944117216204
  1. \(\sum \frac{(-1)^n}{n!}\)
Solución 
error = 10^-10
a(n) = (-1)^n*log(n)/n^2
i = 2
while abs(a(i)) >= error
    i += 1
end
println("Suma parcial de orden $i")
println("Aproximación: $(sum(a, 2:i))")
Suma parcial de orden 357592
Aproximación: 0.10131657821350423

Ejercicio 6.6 La función de Bessel de orden 0 se obtiene a partir de la suma de la serie de potencias

\[ J_0(x) = \sum_{n=0}^\infty (-1)^n\frac{x^{2n}}{2^{2n}(n!)^2} \]

Esta función sirve, entre otras cosas, para explicar la distribución de temperaturas en una lámina circular o las vibraciones de una membrana.

  1. Calcular el radio de convergencia de la función de Bessel.
Ayuda

Utilizar el criterio de la razón para determinar el radio de convergencia de la serie de potencias.

Solución 
using SymPy
@vars n int=true
c(n) = (-1)^n*1/(2^(2n)*factorial(n)^2)
limit(abs(c(n)/c(n+1)), n => Inf)

\(\infty\)

La serie converge para todo \(\mathbb{R}\).

  1. Dibujar en una misma gráfica la suma funcional parcial de orden 10 de la serie y la función de Bessel de orden 0.
Ayuda

Usar el paquete Bessels para dibujar la función de Bessel de orden 0.

Solución 
using Plots, LaTeXStrings, Bessels
@vars x real=true
a(x,n) = (-1)^n/(2^(2n)*factorial(n)^2) * x^(2n)
N = 10

An = sum(an)
plot(An, -8, 8,  ylims = (-1,1), label=L"$\sum_{n=0}^{10} (-1)^n\frac{x^{2n}}{2^{2n}(n!)^2}$")
plot!(besselj0, label = L"$J_0(x)$")

La serie converge para todo \(\mathbb{R}\).

Ejercicio 6.7 Una serie de Fourier es una serie cuyo término general se construye combinando funciones sinusoidales simples (como senos y cosenos) que converge puntualmente a una función periódica continua. La forma general de una serie de Fourier es

\[ a_0 + \sum a_n\cos\left(\frac{2n\pi}{T}t\right)+b_n\operatorname{sen}\left(\frac{2n\pi}{T}t\right), \] donde \(a_n\) y \(b_n\) son los coeficientes de Fourier y \(T\) es el periodo.

Las series de Fourier son muy útiles para aproximar funciones periódicas que describen las ondas, por lo que se utilizan a menudo para modelizar procesos que involucran sonido, imágenes o corriente eléctrica.

Dibujar las gráficas las 10 primeras sumas parciales de las siguientes series de Fourier y predecir hacia qué función convergen.

  1. \(\sum \operatorname{sen}(nt)\)
Solución 
using Plots, SymPy, Latexify
@vars t real=true
a(t,n) = sin(n*t)
N = 10
an = [a(t,n) for n=1:N]
An = cumsum(an)
plots = []  # Array para guardar las gráficas
for i in An
    push!(plots, plot(i, label=latexify(i), legend=:outertop))
end
plot(plots..., layout=(5,2), size=(800,1600))
